[[[ 위키 키위란 무엇인가? ]]]
``위키키위'' 는 작고 간단하고 마음대로 할 수 있는 위키위키(Wiki Wiki) 및 블로그 엔진 입니다. (그럼, " [ 위키위키는 뭔가? ] " 가 궁금하시면 링크를 누르십시오!) 위키키위는 PHP 만 있으면 어디서건 쉽게 돌릴 수 있습니다.

위키키위를 사용하기 위해 할 일은 `` 오직 PHP 가 돌아가고 있는 웹서버에 위키키위를 업로드 '' 하는 것 뿐입니다! 최악의 경우 위키키위는 다른 어떤 것도 필요없이 오직 index.php 파일 딱 하나를 업로드하는 것 만으로 끝을 낼 수 있습니다.

[[[---
 ``  ``` 위키키위의 특징 '''  '' 
 * 오직 PHP만 있으면 어디서든 된다.
 * 디자인을 마음대로 간편하게 바꿀 수 있다. (모인모인 클론, 노스모크 클론 디자인 기본지원)
 *   (!)   [ 블로그 ] 로 사용할 수 있다. (트랙백, RSS 지원)
 *   (!)  어느 브라우저에서건 표준 Javascript 만으로 돌아가는 ```  [ 오에카키 ]  ''' 를 지원한다.

 *  [ 정적 HTML ]  을 생성할 수 있어서, 웹페이지를 급히 만들 때 렌더링 유틸리티로 쓸 수 있다.
 * 페이지에 암호 걸기 기능, 답글 기능이 있다.
 * 매우 간단하면서도 파일업로드/검색 기능을 지원한다.
---]]]

위키키위를 사용하기 위해서는 MySQL 등등의 데이터 베이스 설정이나 Python, Perl 등의 언어 처리 프로그램을 전혀 추가로 설치할 필요가 "CR 없습니다. RC" 여러분이 할 일은 오직 PHP 스크립트를 돌릴 수 있는 웹서버에 위키키위 프로그램을 업로드하면 됩니다. 그걸로 ``끝'' 입니다.

더군다나 여러분은 HTML에 대한 지식만 있으면 위키키위의 디자인을 마음껏 바꿀 수 있습니다. 복잡한 "CR SQL, Python, Perl 등을 설치할 필요도 알 필요도 없습니다. RC"

위키키위의 이름이 위키키위인 것은 이 위키위키가 뉴질랜드에서 처음으로 만들어졌기 때문입니다. 잘 알려져 있다싶히, 키위는 뉴질랜드가 원산이 가장 맛있는 과일의 이름이고, 또 뉴질랜드에서 사는 귀여운 새의 이름이자, 뉴질랜드인들의 별명입니다.


[[[ 위키 키위는 어떻게 사용할 수 있나? ]]]
만약 여러분이 아무런 준비가 되지 않은 평범한 마이크로소프트 윈도우즈 사용자로서 위키위키를 사용하시고 싶으시다면, 위키키위는 최선의 선택이 될 수 있습니다. 여러분은 상용 웹호스팅 업체에서 PHP 스크립트가 가능한 계정을 하나 신청해서 받으신 다음 거기에 압축을 푼 위키키위를 업로드 하기만 하면 됩니다. 그걸로 모든 것이 끝납니다.
 
 * 위키키위 다운로드 사이트 - [http://kldp.net/projects/wikikiwi]

만약 여러분이 스스로 웹서버를 가동하고 싶으시다면 다음 링크를 일단 방문하셔서 추가적인
정보를 얻으시기 바랍니다. 여러분이 해야 할 일은 APACHE 웹서버와 PHP를 설치하시는 것입니다.

 * 웹서버 설치 방법 - [http://my.netian.com/~zzzi/gongjiimg/setup.htm]

설치가 성공적으로 끝나면 해야할 일은 위키키위를 다운로드 받아서 APACHE 웹서버의 계정 폴더에 풀어주는 것 뿐입니다. 아마도 C:/APACHE/HTDOCS 일 가능성이 높습니다. 그러면 여러분은 여러분의 웹서버에 접속하기만하면, 정확하게는 http://주소/index.php 로 접속하기만 하면, 위키키위를 마음껏 사용할 수 있습니다.

위키키위는 다음과 같은 특징을 고려해서 만들어졌습니다.

 * 설치가 간단해야 한다.
 * 사용이 쉬워야 한다.
 * 사용자가 마음대로 간단히 전체 디자인을 마구 바꿀 수 있어야 한다.
 * 기본적인 PHP 이외에는 어떠한 추가적인 복잡한 설정과 설치가 없어야 한다.
 * 위키위키로 만든 페이지를 일반 HTML 페이지로 뜰 수가 있어야 한다.

위키키위의 최대 특징은 세 번째 "사용자가 마음대로 간단히 전체 디자인을 마구 바꿀수 있어야 한다." 입니다. 위키키위는 일반적인 HTML 태그로 이루어진 간단한 HTML 파일인 wk_design.htm을 편집하는 것만으로 디자인을 마음껏 바꿀 수 있게 되어 있습니다.
