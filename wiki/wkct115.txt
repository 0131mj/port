 정적 HTML
UNIX 시간 : 1120370897 / 상용 시간 2005.07.03, 3:08 pm

위키키위는 현재 페이지를 HTML 파일로 "뜰" 수 있습니다. 이 경우에 모든 링크들 역시 해당하는 정적인 HTML 파일에  대한 링크로 변경됩니다.

=== 어떻게 사용하는가? ===
화면의 메뉴중에 ''' Create static HTML ''' 이라는 것을 누르십시오. 그러면, 현재 데이터의 정적 HTML 버전이 생성되어 표시됩니다. 그리고 이것은 페이지 제목.htm 이라는 이름의 파일로 자동으로 서버쪽 파일에 저장됩니다. 모든 링크들 역시 페이지 제목.htm 으로 변환됩니다.

=== 언제 사용하는가? ===
저는 이것을 위키의 렌더링 엔진과 JSOekaki를 이용해서 금방 꾸민 페이지를 다른 곳에 HTML 형식으로 올리거나 붙이기 위해서 사용합니다. 또 위키 키위 페이지들을 백업하기 위해 링크들이 그대로 보관된 채로 데이터들을 오프라인에서 검색하기 간단하게 하기 위해 사용합니다.

=== 조금 더... ===
이렇게 했을 때 만들어진 정적인 HTML 파일은, 업로드, 페이지 수정 등으로 가는 링크는 자동으로 무력화 된 상태가 됩니다. 정적인 HTML 파일을 만들 때, 만약 '' wk_design_static.htm '' 파일이 index.php와 같은 곳에 있으면, 위키키위는 이 파일을 대신 읽어 들여 디자인 파일로 사용합니다. 이로서, 페이지 수정 등의 링크가 없고, 사용 목적에 맞는 다른 깔끔한 디자인을 얻으실 수 있을 겁니다.
